# Cambios de Base

El concepto de **cambio de base** es una herramienta poderosa que permite analizar espacios vectoriales desde diferentes perspectivas. Al cambiar la base, las coordenadas de los vectores cambian, pero la estructura geométrica permanece invariante.

## Objetivos

- Comprender cómo diferentes bases representan los mismos espacios
- Aprender a calcular matrices de cambio de base
- Entender cómo transforman las matrices bajo cambios de base
- Aplicar cambios de base para simplificar problemas

## Contenido

En este capítulo exploraremos:

1. Bases y coordenadas en espacios vectoriales
2. Matriz de cambio de base
3. Transformación de vectores bajo cambio de base
4. Transformación de matrices de transformaciones lineales
5. Diagonalización y formas canónicas
